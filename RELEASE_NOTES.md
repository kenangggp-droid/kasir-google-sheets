# Release Notes

## v1.0.0

- Login berbasis sheet `Users`.
- Dashboard ringkasan penjualan harian dan stok menipis.
- CRUD ringan stok barang pada sheet `Barang`.
- Transaksi dan keranjang belanja.
- Checkout dengan diskon, pajak, metode bayar, update stok, detail invoice, dan catatan kas.
- Riwayat transaksi dari sheet `Transaksi`.
- Backend Google Apps Script sebagai REST API untuk Google Sheets.
